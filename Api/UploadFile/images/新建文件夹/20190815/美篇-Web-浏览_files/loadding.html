九九重阳节 浓浓敬老情——爱美立雅幼儿园大一班重阳节敬老活动

2018.10.17 飘然 阅读 0 音乐

音乐
农历九月初九重阳节是我国的传统节日，又叫“老人节”，是一个属于敬老爱老的日子。在这个特殊的日子里，为了让幼儿感受中华传统美德，让孩子们体验与老人之间浓浓的亲情，并用自己的行动来表达对爷爷奶奶的情感，爱美立雅幼儿园开展了以“尊老、感恩”为目的的主题活动，创建一个爱老敬老养老助老的氛围，通过这些活动，让幼儿懂得去爱自己身边的每一位老人，帮助弱者，在生活中自然地走进感情的世界，体验幸福生活的喜悦，同时也让老人们为自己孩子的成长与进步感到欣慰与骄傲。

有的孩子亲手为爷爷奶奶制作的鲜花，有的孩子为爷爷奶奶画了一幅画，这些都寄托了孩子们浓浓的心意！❤️❤️❤️

小时候婆婆爷爷们总是呵护我们，担心我们着凉，关心我们，现在我慢慢长大了懂事了，让我们在她们的节日里为婆婆爷爷们捶捶背、敲敲腿。宝贝们正在为爷爷奶奶捶背呢！看着爷爷奶奶满意的笑容，心里感到的是温馨、是欣慰！我们的宝贝在成长、在懂事、在长大！心里也是满满的暖意❤️

在温暖感人的氛围中，此次活动圆满落幕。在"九九重阳节，浓浓敬老情"活动中，让老人走上舞台绽放精彩，让老人坐下来享受孩子的尊敬和爱戴，活动中，很多家长热泪盈眶，活动后，很多家长心潮澎湃。我们共同传承中华传统美德，在孩子心中播下一颗敬老、孝老的种子。

祝天下所有老人长寿安康，幸福久长！
